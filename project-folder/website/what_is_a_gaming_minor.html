What is a computer gaming minor

The minor in computer gaming creates a focus for students interested in the practical, social, and theoretical aspects of computer/video games. The core requirements of this minor introduce the primary methods and concepts that are key to the production of computer/video games.

Some of the things I do in this Minor are

  1. Using software to create animated work
  2. Using code to create software to interact and play with
  3. Creating unique styles of artwork through computer generated programs
